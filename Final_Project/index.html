Accessibility Review

Conducted on Univeristy of Michigan's Notable Alumni Website

Designers

  • Make sure there is enough contrast between text and its background color
  • Pair values of colors together (not only hues) to increase contrast
  • Design focus states to help users navigate and understand where they are

    • Color Contrast

    There was a sufficient amount of color contrast lacking in the text on the flyout menu in the upper right corner and on the '+' icons that inform about the expanding menus:

    The colors used are so faint that is very easy to miss these visual indicators on the screen. Increasing the color contrast would help those who suffer from visual imparements to better recognize these signifiers that let them operate the site. Additionally, it would help to enhance the visual design, giving a greater sense of balance and cohesion.

    Tab & Focus

    A big issue on this site was that users cannot tab through the main content if any of the expandable menus are closed. As it is currently designed, users with limited mobility are not able to access the main content by the use of their keyboard.

    I highly recommend implementing more detailed tab-index functionality within the html code that will allow users to navigate to the main content. More specifically, there should be a tab function that would allow users to access the collapsable menus and expand them.

    The designer could go an extra mile to further increase accessibility by implementing the pseudo class :focus in order to bring additional attention to tabbed items. For example, the items in focus could be gently highlighted with a faint/transparent red color. This would create enough color and grayscale contrast to draw attention to the element while simultaneously remaining unabraisive to the eye and causing minimal disturbance to the overall visual design.

    Tools

    Use these tools to help make your work accessible:

    • WCAG 2.0 guidelines to review best practices
    • WAVE (Web Accessibility Evaluation Tool) Chrome Extension to check accessibility across your products
    • Web Accessibility Checker to check accessibility across your products
    • Color Contrast Analyzer App to analyze color contrast
    • Color Palette Accessibility Checker to automatically check color combinations for passing contrast
    • Color Extractor bookmarklet to find all colors in CSS and load them in the Color Palette Accessibility Checker
    • Web Accessibility Checker to check accessibility across your products
    • Color Safe to make accessible color palettes
    • Color Oracle to simulate color blindness
    • Sim Daltonism to simulate color blindness
    • Text over images accessibility tool
